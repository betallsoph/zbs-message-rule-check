# Docs1 - ZBS Rule Check Submission Notes

Folder này là bản giải thích để đọc/nộp cùng Challenge 2.

Nếu đang gấp, đọc 3 file này trước:

1. `06-ban-de-doc-de-hieu.md` - bản giải thích cho mình đọc để hiểu bài.
2. `07-script-noi-voi-giam-khao.md` - bản nói miệng với giám khảo nếu bị hỏi.
3. `08-giai-thich-readme-that-chi-tiet.md` - giải thích cực kỹ từng phần trong README.

Các file nền phía dưới để audit chi tiết hơn:

1. `01-de-bai-va-chien-luoc.md` - đề bài yêu cầu gì và vì sao không làm hết mọi rule.
2. `02-rule-map-de-hieu.md` - rule map bằng mã nội bộ `G/P/S/H`, có nguồn Zalo đi kèm.
3. `03-tool-flow.md` - tool đọc input và chạy check như thế nào.
4. `04-sample-review.md` - đối chiếu sample trong file Excel với output/tool logic.
5. `05-ai-usage-note.md` - ghi chú ngắn về cách dùng AI.
